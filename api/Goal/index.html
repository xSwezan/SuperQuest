Goal

A Goal can be added to a Quest, Goals signifies if the Quest is finished. When all Goals has reached their EndValue they will be detected as finished.

Goals has a StartValue and an EndValue which can be set by the Developer. The Developer can set the Value of a Goal whenever, E.g if a Player mines a Rock in your game, you want that to register in your Quests and add 1 to the Rock Goal. 

local SuperQuest = require(PARENT.SuperQuest)

-- Create a Goal
local SomeGoal = SuperQuest:CreateGoal("SomeGoal", 0, 10)
